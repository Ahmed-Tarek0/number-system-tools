# 🔢 Number System Tools

A Python-based toolkit that provides:

* Number system conversions (Binary, Decimal, Octal, Hexadecimal)
* Binary arithmetic operations:

  * One's Complement
  * Two's Complement
  * Addition
  * Subtraction

---

## 🚀 Features

### 1. Number System Converter

Convert numbers between:

* Decimal
* Binary
* Octal
* Hexadecimal

### 2. Binary Calculator

Perform operations on binary numbers:

* One's complement
* Two's complement
* Binary addition
* Binary subtraction

---

## 🛠️ How to Run

1. Make sure you have Python installed.
2. Clone the repository:

```bash
git clone https://github.com/your-username/number-system-tools.git
```

3. Navigate to the project folder:

```bash
cd number-system-tools
```

4. Run any file:

```bash
python binary_calculator.py
```

or

```bash
python number_system_converter.py
```

## 📌 Example

### Binary Addition

```
Input:
1010 + 0011

Output:
1101
```

### Number Conversion

```
Input:
Decimal: 55

Output:
Binary: 110111
Octal: 67
Hexadecimal: 37
```
